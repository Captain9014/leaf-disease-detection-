# Multi-Crop Plant Disease Detection and Classification Using Deep Learning with Mobile-Based Web Application

## Project Overview
This project is a Deep Learning based web application that detects and classifies diseases in multiple crops using plant leaf images. Users can upload an image of a plant leaf through a mobile-friendly web application, and the system predicts the disease present in the crop.

## Features
- Detect diseases from plant leaf images.
- Supports multiple crop types.
- Deep Learning based prediction.
- Mobile-friendly web interface.
- User image upload facility.
- Displays predicted disease results.

## Technologies Used
- Python
- Django
- TensorFlow / Keras
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

## Dataset
The dataset used for training the model consists of plant leaf images of different crops and diseases.
