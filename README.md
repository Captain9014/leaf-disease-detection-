# Multi-Crop Plant Disease Detection and Classification Using Deep Learning

## Project Overview

This project is a Deep Learning based web application that detects and classifies diseases in multiple crops using plant leaf images. Users can upload an image of a plant leaf, and the system predicts the disease present in the crop.

## Features

* Detects diseases from plant leaf images.
* Supports multiple crop types.
* Deep Learning based disease prediction.
* User-friendly web interface.
* Upload image and get instant prediction results.

## Technologies Used

* Python
* Flask
* TensorFlow / Keras
* HTML
* CSS
* Bootstrap
* SQLite

## Dataset

The dataset consists of images of healthy and diseased plant leaves of different crops such as Potato and Tomato.

## Project Structure

* `app.py` - Main Flask application
* `train_model.py` - Model training script
* `model/plant_model.h5` - Trained deep learning model
* `templates/` - HTML templates
* `static/` - Static files (CSS, Images)
* `dataset/` - Plant disease image dataset
